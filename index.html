BGP Network Engineer Reference

Basic BGP Configuration

router bgp 
bgp log-neighbor-changes
bgp router-id 
neighbor  remote-as 
neighbor  description 
neighbor  update-source 
neighbor  ebgp-multihop 
neighbor  password

Address Family

address-family ipv4
neighbor  activate
neighbor  next-hop-self
neighbor  route-map  in
neighbor  route-map  out
neighbor  prefix-list  in
neighbor  prefix-list  out
exit-address-family

Route Advertisement

network  mask 
aggregate-address   summary-only
redistribute connected
redistribute static

Traffic Engineering

route-map SET-LOCALPREF permit 10
set local-preference 200

route-map SET-MED permit 10
set metric 50

route-map PREPEND permit 10
set as-path prepend

Communities

route-map SET-COMMUNITY permit 10
set community 
neighbor  send-community
neighbor  send-community both

Real ISP BGP Troubleshooting Scenarios

  • BGP neighbor stuck in Idle state
  • BGP neighbor stuck in Active state
  • BGP session flapping intermittently
  • Neighbor established but no prefixes received
  • Prefix received but not installed in routing table
  • Customer cannot reach internet although BGP is up
  • Default route not advertised to downstream customer
  • Full internet table not received from upstream
  • Only partial routes received from provider
  • Traffic going through wrong upstream ISP
  • Route advertised but filtered by prefix list
  • Route-map denying expected prefixes
  • Maximum prefix limit exceeded shutting BGP session
  • AS path prepend not affecting inbound traffic
  • MED influencing wrong exit path
  • Local preference misconfiguration inside ISP network
  • Next-hop unreachable for received BGP routes
  • iBGP route not propagated to other routers
  • Route reflector not reflecting routes
  • Cluster ID conflict in route reflector design
  • BGP dampening suppressing customer routes
  • Frequent route withdrawals causing instability
  • High CPU due to excessive BGP updates
  • MPLS L3VPN route not imported into VRF
  • VPNv4 routes not exchanged between PE routers
