BGP ISP Command Reference

1. Basic ISP BGP Process

router bgp <ASN>
bgp log-neighbor-changes
bgp router-id <router-id>
no bgp default ipv4-unicast
timers bgp 30 90

2. Neighbor Configuration

neighbor <IP> remote-as <ASN>
neighbor <IP> description <PEER-NAME>
neighbor <IP> update-source <interface>
neighbor <IP> password <password>
neighbor <IP> ebgp-multihop <hops>
neighbor <IP> send-community
neighbor <IP> send-community both
neighbor <IP> soft-reconfiguration inbound

3. Address Family Activation

address-family ipv4
neighbor <IP> activate
neighbor <IP> next-hop-self
neighbor <IP> route-map <MAP-NAME> in
neighbor <IP> route-map <MAP-NAME> out
neighbor <IP> prefix-list <LIST-NAME> in
neighbor <IP> prefix-list <LIST-NAME> out
exit-address-family

4. Route Advertisement

network <network-ip> mask <subnet-mask>
aggregate-address <network> <mask> summary-only
redistribute connected
redistribute static

5. Local Preference

route-map SET-LOCALPREF permit 10
set local-preference 200

6. MED

route-map SET-MED permit 10
set metric 50

7. AS Path Prepending

route-map PREPEND permit 10
set as-path prepend <ASN> <ASN> <ASN>

8. Communities

route-map SET-COMMUNITY permit 10
set community <ASN:VALUE>
neighbor <IP> send-community
neighbor <IP> send-community both

9. Prefix Filtering

ip prefix-list CUSTOMER-IN permit <prefix>/<length>
ip prefix-list CUSTOMER-IN deny 0.0.0.0/0 le 32

10. Maximum Prefix Protection

neighbor <IP> maximum-prefix 1000

11. Route Reflector

neighbor <IP> route-reflector-client
bgp cluster-id <cluster-id>

12. BGP Dampening

bgp dampening

13. Default Route Advertisement

neighbor <IP> default-originate

14. BGP Authentication

neighbor <IP> password <PASSWORD>

15. BFD Fast Failover

neighbor <IP> fall-over bfd

16. Verification Commands

show ip bgp summary
show ip bgp
show ip bgp neighbors
show ip bgp neighbors <IP> advertised-routes
show ip bgp neighbors <IP> received-routes
show ip bgp regexp
show ip bgp community
show ip bgp dampening parameters
show ip bgp flap-statistics

17. Troubleshooting

debug ip bgp
debug ip bgp updates
clear ip bgp *
clear ip bgp <neighbor-ip>
clear ip bgp <neighbor-ip> soft in
clear ip bgp <neighbor-ip> soft out
